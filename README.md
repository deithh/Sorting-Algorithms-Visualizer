# Visual-sort
Curious about how computers organize data? This Sorting Algorithm Visualizer offers an engaging way to explore the inner workings of various sorting algorithms. From the methodical Bubble Sort to the speedy Quick Sort, watch as these algorithms sort through data with precision and efficiencyâ€”one step at a time.  
This project is my implementation of programs used in mesmerizing videos i saw on youTube 

install requirements.txt  
In config set array length and window properties  

controls:  
i - generate array  
q - quick sort  
s - insertion sort  
m - mergesort  
esc - stop
